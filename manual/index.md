  - title: Signal-First Reactivity
    details: Signals form a dependency graph. The runtime schedules minimal recomputation automatically.
  - title: GTK Native
    details: First-class JSX-like markup targets GTK4 and libadwaita. No web runtime, no Electron.
  - title: Typed Everything
    details: A strong, closed type system with union types, records, domains, and typeclasses. No implicit coercions.
  - title: Explicit Effects
    details: All side effects are channelled through Sources. I/O, timers, HTTP, keyboard — all declared, all typed.
  - title: Pipe Algebra
    details: A rich set of composable pipe operators makes data flow readable and declarative.
